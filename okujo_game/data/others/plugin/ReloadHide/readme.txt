ティラノビルダーV2及びティラノスタジオで開発中に
プレビュー時、左上に表示されるリロードボタンの外枠部分をクリックすると
非表示にしたり表示させたりできるようになるだけのプラグインです。


■導入方法

このテキストと、init.ksが入っている「ReloadHide」フォルダを
使用するプロジェクトフォルダの「data/others/plugin」
にコピーしてください。
(「data/others/plugin/ReloadHide/init.ks」　となるように配置してください。)

first.ks、またはその直後に読み込むksファイル
（ビルダーの場合は[data/scenario/system/plugin.ks]の1行目を推奨）などの
ゲーム起動時に必ず通過するシナリオファイルで

[plugin name="ReloadHide"]

と記述するとリロードボタンの外枠をクリックすることで非表示、
非表示状態でリロードボタンのあった位置を再びクリックすると再表示できるようになります。


■ゲームリリース時

残しておいても恐らく害はありませんが、気になる場合は
リリース時に[plugin name="ReloadHide"]を削除やコメント化してください。
